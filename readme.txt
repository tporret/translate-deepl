Enterprise-grade DeepL translation integration for WordPress posts and pages.

== Description ==
Translate DeepL provides asynchronous post and page translation using DeepL, with Gutenberg-aware content processing and language-based URL routing.

== Installation ==
1. Upload the plugin folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. Configure API key and active languages under Settings > Translate DeepL.
